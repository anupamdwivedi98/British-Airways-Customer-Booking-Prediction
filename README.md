✈️ British Airways Customer Booking Prediction

📌 Project Overview

This project focuses on predicting whether a customer will complete a flight booking using machine learning techniques.

📊 Dataset

- 50,000+ customer records
- Features include:
  - Trip type
  - Purchase lead time
  - Flight duration
  - Booking origin

⚙️ What I Did

- Data cleaning and preprocessing
- Built a Random Forest Classification model
- Evaluated model performance

📈 Results

- Achieved ~85% accuracy
- Key features:
  - Purchase lead time
  - Length of stay
  - Flight hour

🛠️ Tech Stack

- Python
- Pandas
- Scikit-learn
- Matplotlib
